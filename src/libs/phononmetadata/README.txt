Phonon w systemie Windows nie oferuje dobrego wsparcia formatu MP3,
nawet jeśli zainstalujesz dodatkowe kodeki. 
W niektórych wersjach systemu windows konieczne jest zainstalowanie dodatkowych kodeków.
Znajdziesz je, wpisując w Google "k-lite codec pack full".

W systemie Linux musisz zadbać o dobry dekoder MP3.
Jeśli korzystasz z gstreamer,
pamiętaj o pluginie gstreamer-fluendo mp3 codec.
